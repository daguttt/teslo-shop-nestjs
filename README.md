# Teslo Shop

## Running the app

1. Clone repository
2. Install dependencies (_pnpm friendly_):

```bash
pnpm i
```

or

```bash
npm i
```

3. Add `.env` file relying on `.env.example`. Change environment variables
4. Host database with Docker:

```bash
docker-compose up -d
```

5. Run the app in development mode:

```bash
nest start -w
```
